***アプリケーション名	***
bosotto

***アプリケーション概要***	
チャットルームでテキストや画像だけでなく、音声を用いたコミュニケーションができる。
また、タイマーを設定することで保存した音声を目覚まし時計の代わりにすることができる。

***URL***
 https://bosotto.herokuapp.com/
